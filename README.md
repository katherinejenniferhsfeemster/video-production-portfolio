# Video Production Portfolio — AI Dataset Assets

**Katherine Feemster** · Video Production Professional · [Company, years]

End-to-end, code-first video production for an AI research program: dataset
curation from raw footage, reproducible NLE projects (Shotcut, OpenShot,
Lightworks), interchange to OpenTimelineIO and CMX-3600 EDL, automated captioning
with safe-area burn-in, and a color-managed editorial LUT. Every asset in this
repo is generated by scripts — no manual clicking, no binary leftovers.

## Reproducibility

```bash
# one-shot: build everything from nothing
python3 scripts/python/run_all.py
```

`run_all.py` orchestrates the nine stages:

1. `make_synthetic_clips.py` — three 5 s 1080p30 clips with distinct content (color bars, moving gradient, test pattern) and a 1 kHz tone.
2. `curate_dataset.py` — PySceneDetect + aHash deduplication + `ffprobe` normalization → `assets/data/curated/manifest.csv`.
3. `build_shotcut_mlt.py` — valid Shotcut `.mlt` with profile, three producers, playlist and `dynamicText` filters.
4. `build_openshot_osp.py` — `.osp` JSON and `.otio` OpenTimelineIO file for interchange.
5. `export_lightworks_edl.py` — CMX-3600 `.edl`, a shotlist `.tsv`, and a 33pt `.cube` editorial LUT.
6. `auto_caption.py` — SRT from script (Whisper when `WHISPER=1`), burned with ffmpeg `subtitles` filter.
7. `render_demo.py` — three clips normalized and cross-faded into the final cut.
8. `render_posters.py` — four matplotlib posters for the site.
9. All artefacts land in `scripts/projects/`, `assets/renders/`, `assets/demo/`.

All dependencies are pure FOSS: `ffmpeg`, `opencv-python`, `scenedetect`,
`opentimelineio`, `matplotlib`, `numpy`. Whisper is optional and gated on
`WHISPER=1` so CI runs offline.

## Editorial style

- **Color**: teal `#2E7A7B` + amber `#D9A441` on ink `#0F1A1F` / paper `#FBFAF7`.
- **Type**: Inter (UI) + JetBrains Mono (code / timecode).
- **Safe area**: 5 % title-safe, 10 % action-safe honored in every caption and lower-third.
- **Audio**: EBU R128 integrated loudness target `-23 LUFS`, true-peak `-2 dBTP` (checked with `ffmpeg -filter:a loudnorm=print_format=summary`).
